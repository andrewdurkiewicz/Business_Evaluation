This is a program which exploits the benefits of a block-chain network, and utilizes it to provide for an easy transfer of a homes deed. This purchase is easily done in a quickly done, but is also done safely. Each individual has a unique key hash which is generated independently. Each house also has a unique hash identifier. This allows an easy transaction of keys for owners while keeping an almost perfect aninimity.
